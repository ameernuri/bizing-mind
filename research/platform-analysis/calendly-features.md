# Calendly - Complete Feature Analysis

## Overview
Calendly is a scheduling automation platform focused on eliminating back-and-forth emails for meeting scheduling. It serves individuals, small teams, and enterprises with tiered feature sets.

---

## Core Scheduling Features

### Meeting Types
- **One-on-one meetings** - Individual appointment scheduling (1 on Free, Unlimited on paid)
- **Multi-person meeting types** - Group meetings with multiple participants (paid plans)
- **Round-robin meetings** - Distribute meetings across team members (Teams plan+)
- **Meeting polls** - Find times that work for groups and book the most popular time
- **One-off meetings** - Ad-hoc scheduling without creating permanent event types
- **Managed events** - Admin-created templates synced across teams for consistency

### Calendar Integration
- **Google Calendar** - 1 on Free, 6 on paid plans
- **Microsoft Outlook** - Full integration with Office 365
- **Microsoft Exchange** - Enterprise calendar support
- **iCloud Calendar** - Apple ecosystem support
- **Multi-calendar connections** - Connect up to 6 calendars on paid plans
- **Calendar conflict checking** - Real-time availability checking across all connected calendars

### Availability Management
- **Custom availability hours** - Set specific hours for each day of the week
- **Date-specific hours** - Override availability for specific dates
- **Minimum scheduling notice** - Prevent last-minute bookings (e.g., 24 hours)
- **Buffer time** - Add padding between meetings (e.g., 15 min before/after)
- **Maximum meetings per day** - Limit daily appointment volume
- **Secret event types** - Hidden booking links not indexed publicly
- **Time slot intervals** - Control start times (e.g., every 15, 30, 60 min)

### Video Conferencing Integration
- **Zoom** - Auto-generate meeting links
- **Google Meet** - Automatic link creation
- **Microsoft Teams** - Native integration
- **Webex** - Cisco Webex integration
- **GoTo Meeting** - LogMeIn integration
- **Custom** - Generic video link support
- **In-person** - Physical location meetings
- **Phone call** - Call-in meetings

## Pricing Tiers

### Free ($0)
- 1 event type
- 1 calendar connection
- Unlimited meetings
- Basic customization
- Mobile apps and browser extensions
- Video conferencing integration

### Standard ($10/seat/month)
- Unlimited event types
- Multiple calendars (6)
- HubSpot, Mailchimp connections
- Stripe, PayPal payments
- Zapier, webhooks
- Automated reminders
- 24/7 chat support

### Teams ($16/seat/month)
- Salesforce integration
- Round-robin meetings
- Lead qualification & routing
- Marketo, Pardot forms
- Advanced admin features
- Team analytics
- SSO add-on available

### Enterprise ($15k/year min)
- Salesforce lookup routing
- Microsoft Dynamics integration
- Dedicated account support
- SAML SSO included
- Domain control
- Audit logs
- Data deletion API
- Security reviews

## Strengths
1. **Ease of use** - Minimal setup, intuitive interface
2. **Wide integration** - 100+ integrations available
3. **Scalability** - From individual to enterprise
4. **Workflow automation** - Powerful automation tools
5. **Enterprise security** - Robust security features

## Limitations
1. **Limited payment options** - Only Stripe/PayPal
2. **No inventory management** - Not for product-based businesses
3. **Basic client management** - No full CRM functionality
4. **Limited customization** - Less design flexibility than competitors
5. **No class scheduling** - Individual appointments only (no group classes)
