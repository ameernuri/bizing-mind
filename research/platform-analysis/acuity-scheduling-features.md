# Acuity Scheduling - Complete Feature Analysis

## Overview
Acuity Scheduling is a comprehensive appointment scheduling software owned by Squarespace. It focuses on service-based businesses with robust payment processing, client management, and business operations tools.

---

## Core Scheduling Features

### Online Booking
- **Custom booking website** - Built-in booking page with Squarespace integration
- **Website embedding** - Embed scheduler on existing websites
- **Direct booking links** - Shareable URLs for specific services
- **Multi-location calendar management** - Manage schedules across locations
- **Time zone support** - Automatic timezone detection and conversion (Premium)
- **Custom domains** - Use your own domain for booking page
- **Mobile-responsive** - Works on all devices

### Appointment Types
- **1-on-1 appointments** - Individual client bookings
- **Group classes** - Multiple clients per time slot
- **Workshops and events** - Special event scheduling
- **Recurring appointments** - Regular repeating bookings
- **Packages** - Bundle multiple appointments
- **Subscriptions** - Recurring payment plans for services
- **Waitlists** - Fill cancelled slots automatically

### Availability Controls
- **Custom availability hours** - Per day/week scheduling
- **Date-specific overrides** - Special hours for holidays
- **Service-specific hours** - Different availability per service
- **Buffer times** - Padding between appointments
- **Minimum lead time** - Advance booking requirements
- **Maximum daily limit** - Cap appointments per day
- **Gap reduction** - Optimize schedule to minimize gaps
- **'Look busy' feature** - Hide some availability strategically
- **Fixed start times** - Control when appointments can begin
- **Interval settings** - 15, 30, 60 minute start times

### Resource Management
- **Room booking** - Assign physical spaces
- **Equipment tracking** - Reserve equipment with appointments
- **Station/Chair assignment** - Resource allocation
- **Resource limits** - Prevent double-booking resources

## Strengths
1. **Comprehensive payments** - Full payment processing suite
2. **Package/subscription support** - Advanced billing models
3. **HIPAA compliance** - Healthcare-ready
4. **Squarespace integration** - Seamless website connection
5. **Resource management** - Room/equipment booking
6. **Class scheduling** - Group appointment support

## Limitations
1. **Limited free plan** - No free tier (only 7-day trial)
2. **Squarespace dependency** - Best features with Squarespace
3. **No marketplace** - Less discovery than Mindbody
4. **Limited marketing tools** - Basic email marketing
5. **No branded app** - No custom mobile app option
