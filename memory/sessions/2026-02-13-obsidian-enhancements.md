## Key Obsidian Features Documented

### Built-in (10 features)
1. Graph View — Visual connection explorer
2. Canvas — Infinite visual whiteboard
3. Daily Notes — Temporal journaling
4. Backlinks — "What links here?"
5. Outline — Document structure
6. Tags Pane — Filter by tag
7. Search — Full-text find
8. Command Palette — Quick actions
9. Workspaces — Saved layouts
10. Bookmarks — Quick access

### Community Plugins (10 plugins)
1. Breadcrumbs — Hierarchical navigation
2. Journey — Find paths between notes
3. Supercharged Links — Styled links
4. Metadata Menu — Forms for frontmatter
5. QuickAdd — Rapid capture
6. Note Refactor — Split/combine notes
7. Mind Map — Outline to visual
8. Excalidraw — Hand-drawn diagrams
9. Admonitions — Callout boxes
10. Periodic Notes — Daily/weekly/monthly
