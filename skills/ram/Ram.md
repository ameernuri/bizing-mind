## What Is RAM?

**RAM = Working Memory** ‚Äî the active, immediate context of the mind.

- **Fast access** ‚Äî What's important right now
- **Active processes** ‚Äî What we're working on now
- **Limited space** ‚Äî Keep it tight, focused
- **Temporary** ‚Äî When done, archive or purge

**RAM is NOT:**
- ‚ùå A log (use `memory/sessions/` for logs)

## Usage

### When to Write to RAM

**Add when:**
- Starting new work that matters NOW
- Something needs urgent attention
- Just completed something important (keep brief)
- Blockers that are slowing work NOW
- Context that must not be lost

**Don't add:**
- Routine tasks
- Completed work from >48h ago
- Things that can be looked up elsewhere
- Everything ‚Äî be selective

### When to Remove from RAM

**Remove when:**
- Task is complete ‚Üí archive if significant
- Item is >48h old
- Context is no longer relevant
- Blockers are resolved

**Rule:** RAM should fit on one screen. If it's getting long, archive.
